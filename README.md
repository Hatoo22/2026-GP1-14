# Dir’ Platform

## Introduction
Dir’a is an intelligent web-based platform designed to analyze online video games and detect potential risks within gaming environments. The system focuses on identifying harmful behaviors such as cyberbullying, harassment, hate speech, and other toxic interactions by analyzing user-generated content like reviews and comments.

The goal of the platform is to provide a safer gaming environment, especially for children and adolescents, by offering clear risk insights and early warnings.


## Problem
Modern online games have become social platforms where players interact in real time. While this enhances engagement, it also exposes users to harmful behaviors such as:

Cyberbullying
Harassment
Hate speech
Toxic communication

Current monitoring systems are mostly reactive and depend on user reports, which delays response and increases risk exposure.


## Solution

Dir’a solves this problem by using Artificial Intelligence (AI) and Natural Language Processing (NLP) to:

Analyze large volumes of game reviews and comments
Detect harmful patterns automatically
Classify risks into categories (bullying, threats, etc.)
Assign a dynamic risk score for each game

This enables proactive monitoring instead of reactive reporting.


## Technologies Used
- Python
- Natural Language Processing (NLP)
- Machine Learning
- Transformers
- Pandas


## Users
The platform is designed for:
- Parents & individual users → to choose safer games
- Government authorities → to monitor risks
- Admins → to manage the system


## launch instructions
1. Download the labeled dataset from the `Labeled_Data` folder  
2. Open the model script in the `Model_Training` folder  
3. Run the script to analyze game comments


## Conclusion
Dir’a transforms scattered user data into structured insights using AI, enabling early detection of harmful gaming environments. The platform supports safer digital experiences by shifting from reactive monitoring to proactive risk detection.
